**[APEX](https://github.com/jonathan-vella/apex)**—the *Agentic Platform Engineering eXperience for Azure*—turns plain-language infrastructure intent into reviewed, deployable **Bicep or Terraform**.

**Intent → Requirements → Architecture → Governance → IaC → Deployment → Documentation**

What makes it different:

- Human approval gates before consequential actions
- Independent challenger agents that critique requirements, architecture, governance, and implementation
- Azure Well-Architected Framework assessment and live cost analysis
- Bicep and Terraform execution tracks
- Live Azure Policy discovery and governance-aware planning
- AVM-first code generation with build, lint, security, and deployment validation

The [live walkthrough](https://apexops.pro/demo/) follows a real workload through the complete pipeline in 30 minutes. It preserves the generated artifacts, policy findings, validation trail, adversarial reviews, and deployment evidence—not just a polished product narrative.

**Current Bicep E2E benchmark: 88/100 · Grade B**
